F.A.Q

Frequently Asked Questions

  • What are the different types of coding jobs?

    Web developers: They write the code for a website's looks and functionality.
    Software programmers: They are responsible for creating and making software better.
    Data scientists: They typically work in academia and analyze large amounts of data.
    Mobile app developers: They are responsible for creating mobile software.

  • What is a full stack?

    A Full Stack is made up of the front-end, the back-end, and the database. The front end is what you see when you go onto the internet: the colours, the buttons, the content, etc. Each full stack uses different tools, frameworks, and technologies.

  • What is the most popular programming language?

    JavaScript. JavaScript is very popular because you can use it for multiple different things such as interactive websites, building applications, developing games and lots of other stuff.

  • Is coding really that easy?

    Yes. Coding is quite easy. Especially if you are learning HTML or CSS. This can also depend on the fact the if you have had any interaction with coding before you started learning. For example, a person who know how programming languages work and may find it easier to code than some that have no idea.

  • What languages are used to create operating systems?

    Most operating systems are written in C/C++. But since Apple Created Swift, Swift is also used in a lot of Apple's different operating systems.

  • Who is the best programmers in the world?

    There are tons of gifted programmers in the world (including me) who are very good at coding. I think the best programmer in the world is Tim Berners-Lee since he is the inventor of the World Wide Web which is the entire literally internet.

  • Difference between a designer and a developer?

    While there doesn’t have to be a stark divide between designers and developer (designers can certainly have developer skills and vice versa), designers generally focus on the look and feel of a website while developers focus on making that design come to life.

  • Difference between coding and programming?

    Coding is a process by which we tell computers what to do—how a program should work, how a website should run, etc. Coders use programming languages to communicate these instructions. Nowadays, programming and coding are mostly used interchangeably.

  • How do you get your website online?

    Designing and developing your website is the heavy lifting part. Once design and development are all set, websites go live through the following steps:
    Pick a domain name.
    Register your domain name.
    Choose a web hosting company. (I use Netlify)
    Upload your website.

  • Should I learn to code a site from scratch instead of using Squarespace or Wix?

    While services like Squarespace offer tons of templates and options for customization, your flexibility is limited to what the service you’re using has to offer. This can be fine for projects that require simple or fixed functionality, but to have control over your work and to expand your range of options, you’re going to have to know how to code from scratch.
